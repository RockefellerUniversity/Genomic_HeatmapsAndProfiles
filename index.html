Genomic Heatmaps with profileplyr and Deeptools

The Course

This course introduces plotting and importing Genomic Heatmaps in R.

The course consists of 1 sections on plotting and importing Genomic Heatmaps in R.
Each section is presented as HTMl presentations or single page document. Exercises and answer sheets are included after all subsections to practice techniques and provide future reference examples.

Course material and exercises are available to view as rendered HTML at https://rockefelleruniversity.github.io/Genomic_HeatmapsAndProfiles/.
All material is available to download under GPL v2 license.

Setting up.

Install R.

R can be installed from the R-project website.
R 3.6.0 or higher is required for this course.

http://www.r-project.org/

Install RStudio.

RStudio can be installed from the R-project website.

http://www.rstudio.com/

Install required packages.

Option 1 - For anywhere

Having downloaded R and RStudio, some additional packages are required (profileplyr,GEOquery and ggplot2).
To install these,

  • First launch RStudio
  • Install the packages in the R console using install.packages 
    install.packages("ggplot2",dependencies=TRUE)
install.packages("BiocManager")
BiocManager::install("profileplyr")
BiocManager::install("GEOquery")

Download the material

The material can either be downloaded as a zip

wget https://github.com/rockefelleruniversity/Genomic_HeatmapsAndProfiles/archive/master.zip ./
